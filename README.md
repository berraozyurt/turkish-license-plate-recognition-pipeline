# Turkish License Plate Recognition Pipeline

Independent computer vision project in progress.

## Planned Pipeline

* Turkish license plate detection using Ultralytics YOLO models
* Comparison of lightweight detector variants under the same evaluation protocol
* Plate crop extraction
* OCR-based plate text recognition
* Privacy-safe evaluation and documentation

## Project Scope

This repository is being developed independently using public or synthetic resources. No company code, internship data, private vehicle images, readable real plate identifiers, model weights or client-specific information will be included in the public repository.
